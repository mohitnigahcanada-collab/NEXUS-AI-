# nexus-ai

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run start
```

Then open `http://localhost:4000/v1` for the gateway status or `http://localhost:4000` for the dashboard.

This project was created using `bun init` in bun v1.3.14. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
